## Feature
| 方法 | 作用 |
| --- | --- |
| encodeParamsURI | URL上的params加密方法 |
| decodeParamsURI | URL上的params解密方法 |
| encodeHashURI | URL上的hash加密方法 |
| decodeHashURI | URL上的hash解密方法 |


## Usage

```
import {
    encodeParamsURI, decodeParamsURI, encodeHashURI, decodeHashURI
} from "@dpdfe/event-utils";
```
